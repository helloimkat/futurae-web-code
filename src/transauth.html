Transaction Authentication

Futurae offers a comprehensive authentication portfolio. A variety of technologies allow the highest possible flexibility for organisations of all sizes, from startups to large enterprises. The portfolio allows to choose flexibly between the different transaction signing technologies. Futurae offers easy enrollment and migration of mobile devices. Transaction authentication is used in e-commerce, trading and e-banking web or native applications, in highly regulated industries to keep an audit trail of user actions.

Transaction Signing

Confirmation of a transaction (financial or operational) via One-touch or QR code with the possibility to add a PIN, fingerprint or FaceID.

Via the encrypted Transaction Signing technology, Futurae supports compliance on all levels with the highest standard of security. The modular product secures online and mobile transactions (incl. tablets, smartwatches and IOT devices) of end users in financial, healthcare or online retail cases. The products allow flexibility in adding additional security features such as PIN, fingerprint, or FaceID.

The Transaction Signing technology is available as an SDK, to be easily integrated into existing mobile applications, or via the Futurae secure app that offers corporate identity colours and visuals.

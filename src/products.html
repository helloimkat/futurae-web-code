Futurae Authentication Suite

Strong authentication adds an additional layer of security to logins for your applications (mobile, desktop, laptop, smartwatch or IOT) to protect against data and identity theft by remote attackers. This additional layer of authentication supports companies being compliant with regulatory requirements (GDPR, PSD2, HIPAA/HITECH, FSC, etc.).

Futurae offers a comprehensive authentication portfolio allowing end users secure and seamless access to sensitive data. Futurae offers easy enrollment and migration of mobile devices. The products are used for web applications, platforms and customer portals in highly regulated industries where SMS authentication (MTAN) is not considered secure anymore.

While Futurae focuses so much on usability, we are security researchers at heart. All Futurae protocols are formally verfiied and mobile apps, front- and back-gulpend infrastructure and services go through recurrent penetration testing.

State-Of-The-Art Security

Strong, proven, hardware-backed cryptographic standards. 2048-bit RSA keys, 128-bit AES-CBC, HMAC-SHA256. Protocols run on top of TLS1.2 with certificate pinning.

RESTful API

Open and straightforward APIs easy to implement with a clear interface and awesome documentation.

All in one product

Android and iOS, hosted in the heart of the Swiss mountains, Secure App or SDK, management interface and analytics, support via web, email and phone.
